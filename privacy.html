Prompt Vault 隐私政策

最后更新：2026 年 7 月 14 日

Prompt Vault 是一个浏览器提示词管理工具，用于收集、整理、预览、导入、导出并快速复用 AI Prompt。

当前版本不提供云同步账号、不接入服务器、不自动上传用户保存的 Prompt、图片或反馈内容。

我们处理哪些数据

为了实现功能，Prompt Vault 可能在你的浏览器本地保存以下内容：

  • 你主动保存的 Prompt 文本、负面提示词、英文 Prompt。
  • 你在网页中主动选中并通过右键菜单保存的文字内容。
  • 你主动填写的项目、场景、类型、标签、评分、来源链接。
  • 你保存 Prompt 时的当前网页 URL，用于记录 Prompt 来源。
  • 你主动添加的示例图、参考图或图片链接。
  • 用于功能运行的设置项、收藏状态、使用次数、更新时间。
  • 用于来源识别和防冒名分发的产品、作者、工作室、发布编号等元数据。

网站内容与网络记录

当你在网页中选中文字并选择“保存为 Prompt”时，Prompt Vault 会读取你主动选中的网页文字。这属于 Chrome Web Store 所称的“网站内容”。

当你保存 Prompt 时，Prompt Vault 可能保存当前页面 URL，作为 Prompt 来源记录。这属于 Chrome Web Store 所称的“网络记录”。

Prompt Vault 不会持续监控你的浏览历史，不会自动读取未选中的网页正文，也不会把这些内容上传到开发者服务器。

数据保存在哪里

当前版本的数据主要保存在你的浏览器本地存储中，例如 chrome.storage.local。你导出的 JSON 文件会保存到你自己选择的位置。

如果你手动复制、导出、发送或分享数据，相关内容会离开你的浏览器，由你自行决定分享对象和范围。

是否上传到服务器

当前版本不会自动把你的 Prompt、图片、来源链接或反馈内容上传到再野文化工作室、作者或第三方服务器。

如果未来增加云同步、账号登录、在线反馈、AI 生成或付费功能，本隐私政策会在功能上线前更新，并说明新增的数据处理方式。

权限说明

Prompt Vault 可能请求浏览器存储、当前标签页、脚本插入、右键菜单、剪贴板读取等权限。这些权限用于保存 Prompt、在网页中插入 Prompt、从右键菜单保存选中文字、记录来源 URL、读取剪贴板图片或文本等功能。

第三方共享

当前版本不向第三方出售、出租或自动共享你的 Prompt 数据。

用户如何管理数据

  • 你可以在扩展内删除单条 Prompt。
  • 你可以导出 JSON 作为备份。
  • 你可以在浏览器扩展管理页移除扩展，浏览器可能会同时删除扩展本地数据。
