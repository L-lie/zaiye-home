你可以安装并使用 Prompt Vault 来保存、整理、导入、导出和复用自己的 AI Prompt 数据。

  • 删除、遮盖或修改作者、工作室、版权、来源标识。
  • 将扩展改名、换标、重新包装后发布为自己的产品。
  • 倒卖、转售、批量分发或作为竞品主体使用。

Prompt Vault 导出的 JSON 文件可能包含产品名、作者、工作室、官网、发布编号等来源信息。这些信息用于保留归属、辅助识别来源和防止冒名分发。
