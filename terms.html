Prompt Vault 用户协议

最后更新：2026 年 7 月 6 日

使用 Prompt Vault 即表示你理解并同意本协议。Prompt Vault 由田田田野里 / 再野文化工作室制作和发布。

服务内容

Prompt Vault 提供 Prompt 保存、分类管理、搜索、收藏、导入导出、网页内快速插入等功能。当前版本以本地浏览器工具为主，不承诺提供云端存储。

用户内容

你保存、导入、复制或导出的 Prompt、图片、标签、来源链接等内容归你或其合法权利人所有。请确保你有权保存和使用这些内容。

禁止行为

  • 使用本扩展保存或传播违法、侵权、恶意或违反平台规则的内容。
  • 删除版权和来源标识后冒名分发扩展。
  • 未经许可复制、转售、改名发布或商业化分发本扩展。
  • 利用本扩展攻击网站、绕过平台限制或损害他人权益。

数据备份

当前版本主要依赖浏览器本地存储。浏览器清理数据、卸载扩展、设备故障或误操作都可能导致数据丢失。建议你定期导出 JSON 备份。

免责声明

Prompt Vault 按现状提供。作者会尽力改进体验，但不承诺扩展在所有网站、所有浏览器版本或所有设备上都完全可用。

协议更新

如果后续增加云同步、账号系统、AI 生成、付费功能或团队功能，本协议可能会更新。更新后的版本会发布在官网。
